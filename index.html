News

  • A few projects were accepted to ICLR 2022:
    • The Information Geometry of Unsupervised Reinforcement Learning (oral): We prove that unsupervised skill learning algorithms, such as DIAYN, are not optimal for learning all reward-maximizing policies. I think this work is interesting because it suggests that other, yet-uninvented, skill learning algorithms may be much better for preparing to solve new tasks.
    • Maximum Entropy RL (Provably) Solves Some Robust RL Problems: MaxEnt RL is not necessarily better than purpose-designed robust RL methods, but it is very simple and has appealing formal robustness guarantees.
    • C-Planning: An Automatic Curriculum for Learning Goal-Reaching Tasks (with Tianjun Zhang): We apply variational inference to the problem of goal-conditioned RL: the M step corresponds to short-horizon goal reaching ("thinking fast") while the E step corresponds to long-horizon planning ("thinking slow"). The resulting method significantly outperforms prior goal-conditioned RL methods, and does so without requiring any planning at test time.
    • RvS: What is Essential for Offline RL via Supervised Learning? (with Scott Emmons and Ilya Kostrikov): We show that simple conditional behavior cloning performs surprisingly well on offline RL benchmarks, if certain design decisions are made appropriately.
